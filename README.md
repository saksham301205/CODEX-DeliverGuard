## 📌 Overview

> DeliverGuard is a parametric insurance system that protects delivery workers from income loss using automated triggers — no manual claims required.

---

## 🧩 Problem & Persona

### 🚨 Problem

Delivery workers face income loss due to:

- 🌧 Heavy Rain  
- 🌫 High Pollution (AQI)  
- 🌡 Heatwaves  
- 🚫 Curfews  

---

### 👤 Persona

**Ravi — Delivery Partner (Chennai)**  
Daily earnings: ₹800–₹1000  
Disruptions → fewer orders → income loss  

---
